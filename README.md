# git 사용법
(1) add .
- 파일들을 스테이징 영역에 추가
- 아직 GitHub에 올라가진 않음
- 다음 커밋에 포함시킬 준비 완료

(2) git commit -m '메시지'
- 스테이징된 파일들을 로컬 Git 저장소에 저장
- 변경사항에 대한 설명을 추가
- 아직 GitHub에 올라가진 않음

(3) git push
- 로컬 커밋들을 GitHub에 업로드

(4) git status
- 현재 상태 확인

(5) git log
- 커밋 히스토리 보기
